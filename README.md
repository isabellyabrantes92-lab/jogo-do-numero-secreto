## 📌 Descrição do Projeto 
Este projeto consiste em um jogo interativo desenvolvido com JavaScript, onde o objetivo é adivinhar um número secreto gerado aleatoriamente pelo sistema.

## 🔨 Funcionalidades do Projeto

- Geração de número aleatório dentro de um intervalo definido  

- Captura e validação dos dados inseridos pelo usuário
  
- Comparação lógica entre palpite e número secreto
    
- Feedback dinâmico (maior ou menor)
    
- Controle de tentativas realizadas
    
- Exibição de mensagem de sucesso ao acertar
    
- Reinicialização do jogo sem recarregar a página
    
- Interação com a interface por meio da manipulação do DOM

## 🧠 Conceitos aplicados
- Lógica de programação  
- Estruturas condicionais (if/else)  
- Funções em JavaScript  
- Manipulação do DOM  
- Eventos e interação com o usuário  
- Controle de estado da aplicação
